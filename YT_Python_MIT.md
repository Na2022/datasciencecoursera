### Assigning


```
x, y, z = "Orange", "Banana", "Cherry"
print(x)
print(y)
print(z)
```

```
x=y=z="Cherry"
print(x)
print(y)
print(z)
```
### Output syntax
```
x = "Python "
y = "is "
z = "awesome"
print(x + y + z)
```
```
x = 5
y = "John"
print(x, y)
```

### Unpacking
```
fruits = ["ace", "banana", "change"]
a, b, c = fruits
print(a)
print(b)
print(c)
```
